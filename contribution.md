# Contribution Guide

Please see the full contribution guide in [CONTRIBUTING.md](CONTRIBUTING.md).

## Quick Summary

- Fork the repository and create a feature branch.
- Keep changes focused and include tests.
- Run `go test ./...` before submitting.
- Ensure `gofmt` has been applied.
- Open a pull request with a clear description.

## Code of Conduct

Be respectful and constructive. Any abusive or harassing behavior will not be tolerated.
